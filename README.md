# User-Journey-Case-Study
Built a Customer Journey Map (case study) for Google Maps, focused on improving user engagement and retention across Indian tourist destinations.

Problem explored:
Users often struggle to identify the fastest and most reliable routes in real time—especially when traffic conditions, local road patterns, and unfamiliar locations change frequently.

Key opportunities identified:
• Increase trust through route confidence indicators
• Personalize rerouting based on travel context
• Reduce friction during route selection
• Encourage repeat usage through successful trip experiences

This exercise helped connect:
User challenge → First successful outcome → Long-term engagement
